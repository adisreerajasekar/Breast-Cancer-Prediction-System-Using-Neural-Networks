# Breast Cancer Prediction System Using Neural Networks

## Overview
This project focuses on predicting breast cancer using Artificial Neural Networks (ANN) and Machine Learning techniques. The system analyzes medical diagnostic features and predicts whether the tumor is benign or malignant.

The project includes data preprocessing, exploratory data analysis, feature scaling, neural network model development, training, and performance evaluation for accurate breast cancer prediction.

---

## Problem Statement
Early detection of breast cancer is critical for improving treatment outcomes and patient survival rates. Manual diagnosis can be time-consuming and may lead to inconsistencies.

This project aims to develop an intelligent prediction system using Artificial Neural Networks to assist in identifying breast cancer based on medical diagnostic data.

---

## Objectives
- Analyze breast cancer diagnostic data
- Perform data preprocessing and feature scaling
- Build an Artificial Neural Network model
- Train and evaluate the neural network
- Predict whether a tumor is benign or malignant
- Improve prediction accuracy using deep learning techniques

---

## Dataset
The dataset contains medical diagnostic measurements used for breast cancer classification.

### Dataset Features
The dataset includes features such as:
- Radius
- Texture
- Perimeter
- Area
- Smoothness
- Compactness
- Concavity
- Symmetry
- Fractal Dimension

### Target Variable
- `0` → Malignant
- `1` → Benign
